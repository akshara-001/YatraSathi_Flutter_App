# YatraSathi Flutter App

YatraSathi is a Flutter-based travel application that helps users explore
destinations, nearby places, hotels, and travel-related information.

## Features
- Explore popular destinations
- View nearby places
- Travel gallery
- Hotel and restaurant information
- Cross-platform support (Android, iOS, Web, Desktop)

## Tech Stack
- Flutter
- Dart
- REST APIs

## Getting Started

### Prerequisites
- Flutter SDK
- Dart
- Android Studio / VS Code

### Run the App
```bash
flutter pub get
flutter run
